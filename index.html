Nueva Era Digital (StartUp)

17 de Noviembre del 2024 - 15 de Abril del 2026

Inicié como parte de un programa de estadías profesionales y posteriormente continué como desarrollador, asumiendo responsabilidades completas en desarrollo e infraestructura.

  • Optimicé una API en GraphQL para la extracción, normalización y transformación de datos mediante web scraping desde WordPress, habilitando su uso como Headless CMS y reduciendo el tiempo de respuesta en un 80%, mejorando la disponibilidad de datos para múltiples aplicaciones.
  • Contribuí al desarrollo de un sistema CRM multiplataforma, integrando módulos de IA, business intelligence y gestión de pagos dentro de una arquitectura escalable, mejorando el control operativo y la toma de decisiones.
  • Implementé y administré infraestructura basada en contenedores (Docker), optimizando el despliegue, orquestación y mantenimiento de servicios en entornos productivos.
  • Desarrollé una API de procesamiento de video en Python (MoviePy, JWT), automatizando la composición, edición y renderizado de múltiples clips.
  • Participé en el diseño y desarrollo de una plataforma de gestión, ventas y distribución en el sector turístico, simplificando la administración y publicación de productos y servicios.
  • Desarrollé un sistema POS integrado con el CRM, utilizando tecnologías web multiplataforma para garantizar alta interoperabilidad y escalabilidad entre sistemas.
  • Contribuí al desarrollo de una API en Node.js para la ingesta y almacenamiento de conversaciones de WhatsApp, permitiendo el monitoreo, trazabilidad y análisis del desempeño de agentes.
